# ddos_simulation

DDoS simulation written in Python using "scapy" Used for COSI-107A final project

## Options:

There are 3 different DDoS attacks:

- TCP Flood 

- UDP Flood

- ICMP Flood


You can customize the DDos attacks by different options

- IP to attack

- Number of IPs

- Number of messages per IP

- Type of attack

- Origin of IP addresses (Single or Random)


## Requirements
Python 3.x:

```
pip3 install scapy
```
